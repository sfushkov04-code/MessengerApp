# MessengerApp

Простой мессенджер на SwiftUI с использованием Firebase.

## 🚀 Возможности

* Отправка сообщений в реальном времени
* Онлайн-обновление через Firestore
* Анонимная авторизация пользователей
* Разделение сообщений (свои / чужие)

## 🛠 Технологии

* SwiftUI
* Firebase Firestore
* Firebase Auth

## 📱 Как запустить

1. Клонировать репозиторий
2. Открыть в Xcode
3. Запустить на симуляторе или iPhone
